Autor: David Andres Vargas Leon

# Instrucciones 

1. Clone repository::

```
git clone https://github.com/GEARSHORDA3/React-JS-Clones/tree/main/netlflix-clone

```

2. Access to project:

```
cd netlflix-clone
```

3. In cmd compile the project:

```
npm install
npm start
```

4. If error 'react-scripts': 

```
npm install react-scripts --save
```
